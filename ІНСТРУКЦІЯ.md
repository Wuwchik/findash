# 🚀 Інструкція розгортання FinDash на Vercel

## Що вам знадобиться
- Комп'ютер (Windows, Mac або Linux)
- 20-30 хвилин часу
- Безкоштовний акаунт на GitHub та Vercel

---

## КРОК 1 — Встановити Node.js
1. Відкрийте браузер, перейдіть на https://nodejs.org
2. Натисніть велику зелену кнопку **"Download Node.js (LTS)"**
3. Встановіть як звичайну програму (далі → далі → встановити)
4. Після встановлення відкрийте **Термінал** (Mac: Cmd+Space → "Terminal") або **Командний рядок** (Windows: Win+R → "cmd")
5. Введіть команду та натисніть Enter:
   ```
   node --version
   ```
   Якщо бачите щось на кшталт `v20.x.x` — Node.js встановлено ✓

---

## КРОК 2 — Розпакувати та підготувати проєкт
1. Розпакуйте файл `findash.zip` у будь-яку папку на комп'ютері
2. Відкрийте Термінал/Командний рядок
3. Перейдіть у папку проєкту:
   ```
   cd шлях/до/папки/findash
   ```
   Наприклад: `cd Downloads/findash`
4. Встановіть залежності:
   ```
   npm install
   ```
   Зачекайте 1-2 хвилини поки завантажаться пакети
5. Перевірте що все працює локально:
   ```
   npm run dev
   ```
   Відкрийте браузер і перейдіть на http://localhost:5173 — ви побачите FinDash ✓
6. Зупиніть сервер: Ctrl+C

---

## КРОК 3 — Завантажити на GitHub
1. Перейдіть на https://github.com та створіть безкоштовний акаунт (якщо немає)
2. Натисніть **"New repository"** (зелена кнопка)
3. Назвіть репозиторій: `findash`
4. Залиште **Public**, натисніть **"Create repository"**
5. У Терміналі (в папці findash) виконайте по черзі:
   ```
   git init
   git add .
   git commit -m "Initial commit"
   git remote add origin https://github.com/ВАШ_ЛОГІН/findash.git
   git push -u origin main
   ```
   Замініть `ВАШ_ЛОГІН` на ваш логін GitHub

---

## КРОК 4 — Розгорнути на Vercel
1. Перейдіть на https://vercel.com
2. Натисніть **"Sign Up"** → **"Continue with GitHub"** — увійдіть через ваш GitHub акаунт
3. Натисніть **"Add New Project"**
4. Знайдіть репозиторій `findash` у списку → натисніть **"Import"**
5. Налаштування залишіть за замовчуванням (Vercel автоматично визначить Vite)
6. Натисніть **"Deploy"**
7. Зачекайте 1-2 хвилини ⏳
8. Vercel видасть вам посилання типу: **https://findash-username.vercel.app** 🎉

---

## КРОК 5 — Додати на телефон як додаток
### iPhone:
1. Відкрийте Safari (саме Safari, не Chrome)
2. Перейдіть на ваше посилання Vercel
3. Натисніть кнопку "Поділитись" (квадрат зі стрілкою вгору)
4. Виберіть **"На екран «Додому»"**
5. Натисніть **"Додати"**

### Android:
1. Відкрийте Chrome
2. Перейдіть на ваше посилання
3. Натисніть три крапки (меню)
4. Виберіть **"Додати на головний екран"**

## ❓ Часті питання

**Де зберігаються дані?**
У вашому браузері (localStorage). Дані на телефоні — окремо від даних на комп'ютері. Якщо хочете синхронізацію між пристроями — скажіть, додамо хмарну базу даних (безкоштовно через Supabase).

**Як оновити сайт після змін?**
Просто знову виконайте `git add . && git commit -m "update" && git push` — Vercel автоматично оновить сайт.

**Як зробити резервну копію даних?**
В розділі "⬆ Імпорт" є кнопка **"💾 Скачати резервну копію"** — скачує JSON файл з усіма даними.
